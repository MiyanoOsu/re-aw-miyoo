# Re_Another_World
(Out Of The World 1) (Miyoo homebrew)

Port for Powkiddy Q90/V90

build from source: make

*Caution: Quick save before enter password screen, because press Start will clean all progress.

Keymap:

    A: Jump
    B: Run / Shoot / Action
    Y: pause
    X: hack speed
    Select: exit
    Start: Password screen
    L: quick save
    R: quick load
